# trip_type_classification

### Task ###

* Classification of 38 Trip types based on 5 features( department, Upc, FinelineNumber, Weekday, Scan count)

### Feature engineering ###

* Work with returnings
* 10 features aggregated by weekday and visit number
* Ratios/average values based on aggregated features
* Top Upcs and Departments
* Entropies

### Model ###

* XGBoost

### Result ###

* 1.37 log loss
